# B62 Backstage Production Visual Checklist

- Runtime badge appears as Vercel Control Plane.
- Start/stop/resume/recovery/post-match buttons are disabled or unavailable in production.
- Worker status renders when persisted data exists.
- Active sessions render while local worker is running.
- Heartbeat and freshness render.
- Leases render.
- Latest daily report renders when persisted.
- Latest causal cases render when they exist.
- Stale/empty state is clear and not treated as failed.
- Limitations are visible.
- No component shows odds.
- No component suggests stake or auto-bet.
- No component promises accuracy.
- No frontend component calculates pattern, event, governance, outcome, or probability.
